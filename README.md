### 💡 Algorithm Strategy

효율적인 성장을 위해 아래 **6가지 학습 원칙**을 준수합니다.

1. **Time Limit**: 문제당 **30분~1시간** 고민 후 해결되지 않으면 풀이를 참고합니다.
2. **Understand & Retype**: 타인의 풀이는 반드시 **원리를 이해한 뒤 직접 타이핑**하며 체득합니다.
3. **Concept First**: 생소한 알고리즘은 구글링을 통해 **이론을 먼저 정리**하고 풀이에 임합니다.
4. **Triple Practice**: 새로 배운 알고리즘은 유사한 유형으로 **최소 3문제** 이상 풀이합니다.
5. **Review Cycle**: 이해가 부족했던 문제는 **일주일 뒤 반드시 다시** 풀어보며 복습합니다.
6. **Code Review**: 내 풀이와 타인의 풀이를 비교하며 **더 나은 시간/공간 복잡도**를 탐색합니다.
